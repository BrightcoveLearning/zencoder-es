Usar Zencoder con Google Cloud Storage

Este tema explica la configuración para usar Zencoder en Google Cloud Platform.

Configuración de Google Cloud Storage

Genere credenciales GCS interoperables

Advertencia: El soporte para las regiones de Google Compute Engine se suspendió el 30 de mayo de 2017.

Google Cloud Storage es interoperable con la API de Amazon S3, pero requiere un trabajo de configuración inicial. Para comenzar con GCS en Zencoder, deberá habilitar Acceso interoperable desde Google Cloud Console y genere una clave. Revisar la Para generar una clave de desarrollador sección en este documento para obtener instrucciones sobre cómo activar el acceso interoperable y generar claves de acceso.

Credenciales de la tienda

Una vez que tenga un conjunto de credenciales GCS interoperables para su cuenta, puede almacenarlas de forma segura con Zencoder, en el Credenciales de Zencoder página.

En Agregar credenciales, selecciona Google Cloud Storage. Luego, rellene el Google Cloud Storage Access Key ID y Secret Access Key parámetros con sus credenciales recién generadas. Asegúrese de hacer clic en "Credenciales predeterminadas para transferencias con gcs"; esto garantizará que cualquier URL de entrada o salida que especifique la gcs:// el protocolo utilizará estas credenciales.

El siguiente ejemplo muestra un trabajo en funcionamiento que extrae un video de entrada de GCS, crea representaciones para HTTP Live Streaming y envía las transmisiones de salida a GCS. Puede probar esto cambiando my-bucket a un depósito de GCS existente de tu propiedad y pegarlo en el Generador de solicitudes.

insinuación

hint:Boolean

Versiones de API: V1, V2

Padre: producción

Defecto: falso

Valores válidos: verdadero o falso

Tipos de trabajo compatibles: Todas

Ejemplo: cierto

Descripción:

Active la sugerencia de archivos MP4 para RTP / RTSP. El valor por defecto mtu_size es 1450 bytes; utilizar el mtu_size opción para cambiar esto.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "hint": true
    }
  ]
}

mtu_size

mtu_size:Integer

Versiones de API: V1, V2

Padre: producción

Defecto: 1450

Valores válidos: Un número entero entre 100 y 5000

Tipos de trabajo compatibles: Todas

Ejemplo: 2000

Descripción:

Tamaño de MTU en bytes para sugerencias MP4.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "mtu_size": 2000
    }
  ]
}

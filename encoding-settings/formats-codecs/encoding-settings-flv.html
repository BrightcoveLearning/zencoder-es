cue_points

cue_points:Array

Versiones de API: V1, V2

Padre: salidas

Valores válidos: Una variedad de hashes con configuraciones de puntos de referencia

Tipos de trabajo compatibles: vod

Descripción:

Zencoder admite la adición de puntos de referencia de navegación y eventos a los archivos VP6 Flash Video (FLV). Los puntos de referencia están incrustados en el flujo FLV y el paquete de metadatos FLV cuando codifica el archivo FLV.

Hay dos tipos de puntos de referencia: "navegación" y "evento". Los puntos de referencia de navegación se pueden utilizar para permitir que los usuarios busquen una parte específica de un archivo de video. Los puntos de referencia de navegación crean un fotograma clave en la ubicación del punto de referencia especificado, por lo que puede usar código para mover el cabezal de reproducción de un reproductor de video a esa ubicación. Puede establecer puntos particulares en un archivo FLV donde podría desear que los usuarios busquen. Por ejemplo, su video puede tener varios capítulos o segmentos, y puede controlar el video incrustando puntos de referencia de navegación en el archivo de video.

Los puntos de referencia de eventos se pueden usar para realizar acciones en un reproductor de video SWF que se activan en puntos específicos durante la reproducción de FLV.

Para obtener más información, consulte este Artículo de Adobe sobre puntos de referencia.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "cue_points": [
        {
          "type": "navigation",
          "time": 30.4,
          "name": "My Cue Point",
          "data": {
            "mykey": "myvalue"
          }
        }
      ]
    }
  ]
}

Recomendaciones de codificación

AV1 es un formato de codificación de video de próxima generación abierto, libre de regalías de Alliance of Open Media Video. Puede ofrecer una calidad superior a velocidades de bits significativamente más bajas en comparación con los códecs HEVC, VP9 y H.264.

Ver el Guía AV1 para obtener más información, y preste especial atención a la Limitaciones para soporte actual.

Además de las configuraciones que se encuentran en esta página, es posible que desee incluir otras configuraciones generales de video, pero tenga en cuenta que las siguientes configuraciones son no actualmente soportado:

  • video_reference_frames
  • video_bframes
  • max_video_bitrate
  • quality
  • crf
  • decoder_bitrate_cap

Tenga en cuenta también que para la configuración de Resolución (ya sea que utilice size o width y height) tanto la anchura como la altura deben ser múltiplos de 8.

video_codec_level

video_codec_level:String

Versiones de API: V2

Padre: salidas

Valores válidos:

  • 2.0
  • 2.1
  • 3.0
  • 3.1
  • 4.0
  • 4.1
  • 5.0
  • 5.1
  • 5.2
  • 5.3
  • 6.0
  • 6.1
  • 6.2
  • 6.3

Tipos de trabajo compatibles: Todas

Ejemplo: 4.1

Descripción:

Restringe la tasa de bits y las unidades de árbol de codificación (CTU) (AV1).

Tenga en cuenta que esta configuración se puede utilizar para otros códecs con diferentes valores permitidos. Ver Configuración de codificación de video.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_level": "5.1"
    }
  ]
}

video_codec_profile

video_codec_profile:String

Versiones de API: V2

Padre: salidas

Valores válidos: main, high, professional

Defecto: main

Tipos de trabajo compatibles: Todas

Ejemplo: 4.1

Descripción:

Establece el perfil de codificación.

Tenga en cuenta que esta configuración se puede utilizar para otros códecs con diferentes valores permitidos. Ver Configuración de codificación de video.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_profile": "high"
    }
  ]
}

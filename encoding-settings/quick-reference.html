    Ajustes de codificación

    En este tema se proporcionan tablas de todas las configuraciones de codificación con vínculos a guías detalladas de cada configuración.

    Tarea

    Ajuste Predeterminado Descripción
    api_key ninguno La clave de API para su cuenta de Zencoder.
    dynamic_profile_options ninguno Con CAE, el número de salidas y parámetros de dichas salidas puede cambiar dinámicamente para cada archivo de entrada. La configuración de dynamic_profile_options define la forma en que se deben crear las copias.
    generate_dynamic_profile falso Con CAE, el número de salidas y parámetros de dichas salidas puede cambiar dinámicamente para cada archivo de entrada.
    input ninguno Una URL HTTP, S3, Cloud Files, GCS, FTP, FTPS, SFTP, Azure o Aspera donde podemos descargar archivos para transcodificar.
    outputs ninguno Una matriz o hash de la configuración de salida.
    region nosotros Región en la que se procesa un archivo: Estados Unidos, Europa, Asia o Australia.
    test falso Habilite el modo de prueba («Modo de integración») para un trabajo.
    private falso Habilitar el modo de privacidad para un trabajo.
    download_connections 5 Utilice múltiples conexiones simultáneas para acelerar la descarga (en algunas circunstancias).
    pass_through ninguno Información opcional para almacenar junto a este trabajo.
    mock falso Envíe una solicitud de trabajo burlada.
    grouping ninguno Una agrupación de informes para este trabajo.
    aspera_transfer_policy justo Cómo asignar ancho de banda disponible para transferencias de archivos Aspera.
    transfer_minimum_rate 1000 Una velocidad objetivo en Kbps para los mínimos de transferencia de datos.
    transfer_maximum_rate 250000 Una velocidad objetivo en Kbps para máximos de transferencia de datos.
    expected_md5_checksum ninguno La suma de comprobación esperada del archivo de entrada.
    credentials ninguno Referencias a credenciales guardadas por un apodo.

    Configuración general de salida

    Ajuste Predeterminado Descripción
    type estándar El tipo de archivo que se va a generar.
    label ninguno Etiqueta opcional para esta salida.
    dynamic_profile_rendition ninguno La configuración dynamic_profile_rendition especifica que esta salida debe utilizar la configuración del perfil CAE generado dinámicamente, y específicamente la copia del perfil desde la que se utilizará la configuración.
    url ninguno Una URL S3, Cloud Files, GCS, FTP, FTP, SFTP, Aspera, Azure, HTTP o RTMP donde colocaremos el archivo transcodificado.
    secondary_url ninguno Una URL S3, Cloud Files, GCS, FTP, FTP, SFTP, HTTP, RTMP, Aspera o Azure donde colocaremos el archivo transcodificado.
    base_url ninguno Una dirección URL de directorio S3, Cloud Files, GCS, FTP, FTP, SFTP, Azure o Aspera donde pondremos el archivo transcodificado, sin nombre de archivo.
    filename ninguno Nombre de archivo de un archivo terminado.
    package_filename ninguno El nombre de archivo de una salida empaquetada.
    package_format ninguno Formato de comprimido/empaquetado que se utilizará para los archivos de salida.
    device_profile ninguno Un perfil de dispositivo que se va a utilizar para la compatibilidad con dispositivos móviles.
    strict falso Habilite el modo estricto.
    skip_video falso No emite una pista de vídeo.
    skip_audio falso No emite una pista de audio.
    source ninguno Hace referencia a una etiqueta en otro trabajo y utiliza el vídeo creado por esa salida para procesarlo en lugar del archivo de entrada.
    credentials ninguno Referencias a credenciales guardadas por un apodo.
    generate_md5_checksum falso Genere una suma de comprobación MD5 del archivo de salida.
    parallel_upload_limit 30 para S3, 10 para todos los demás destinos. El número máximo de cargas simultáneas que se deben intentar.
    headers ninguno encabezados HTTP para enviar con su archivo cuando lo subamos.

    Configuración de CAE

    Ajuste Predeterminado Descripción
    min_renditions 1 El número mínimo de copias que se generarán.
    The maximum number of renditions to generate 8 El número máximo de copias que se generarán.
    min_resolution 192x108 Resolución mínima a tener en cuenta para la escalera.
    max_resolution resolución de origen Resolución máxima a tener en cuenta para la escalera.
    max_frame_rate 30 Velocidad máxima de fotogramas que se puede utilizar.
    keyframe_rate 0.5 Velocidad de fotogramas clave. Este valor puede ser 0.5 o 1. Para el contenido habilitado para SSAI, este valor debe ser 1.
    fixed_frame_rate true Indica al generador que utilice la misma velocidad de fotogramas en todas las copias. Si es falso, las copias de perfil de línea base H.264 con resoluciones inferiores a 360p se les asignará la velocidad de fotogramas a la mitad.
    min_bitrate 200 Velocidad de bits mínima [kbps] que se puede utilizar.
    max_bitrate 15000 Velocidad de bits máxima [kbps] que se puede utilizar.
    max_first_rendition_bitrate 400 Velocidad de bits máxima [kbps] que se puede utilizar para la primera copia.
    min_granularity 30 Cambio relativo mínimo en la velocidad de bits entre copias.
    max_granularity 90 Cambio relativo máximo en la velocidad de bits entre copias.
    video_codec h264 Códec de vídeo a usar.
    bitrate_cap_to_bitrate_ratio 1.3 Relación entre la velocidad de bits máxima y la velocidad de bits media que se utilizará para la codificación VBR.
    buffer_size_to_bitrate_ratio 2.0 Relación entre el tamaño del búfer del decodificador y la velocidad de bits promedio que se utilizará para la codificación VBR.
    bitrate_cap_offset 64 Constante agregada al calcular la velocidad de bits máxima.
    buffer_size_offset 85 Constante agregada al calcular el tamaño del búfer VBV.
    quality_rate_tune 0 Parámetro de ajuste de calidad vs velocidad.
    select_baseline_profile_configuration true Al menos una copia utilizada en el perfil será perfil de línea base.
    video_configurations Ninguno Un conjunto de estructuras que describen resoluciones, velocidades de fotogramas y configuraciones de códec asociadas que deben generarse preferencial o obligatoriamente como parte de la codificación CAE.

    Formato y códecs

    Ajuste Predeterminado Descripción
    format Determinada por el nombre de archivo de salida y, a continuación, el códec de vídeo o audio. De lo contrario: mp4 (para salidas estándar); ts (para salidas segmentadas). El formato de salida que se va a utilizar.
    video_codec Determinada por el formato, perfil o audio_codec. h264 si no se proporciona ninguno. El códec de vídeo que se va a utilizar.
    audio_codec Determinada por el formato, perfil o video_codec. aac si no se proporciona ninguno. El códec de audio que se va a utilizar.

    Resolución

    Ajuste Predeterminado Descripción
    size ninguno Resolución del vídeo de salida (WxH, en píxeles).
    width ninguno El ancho máximo del vídeo de salida (en píxeles).
    height ninguno La altura máxima del vídeo de salida (en píxeles).
    upscale falso Aumente la salida si la entrada es menor que la resolución de salida de destino.
    aspect_mode preservar Qué hacer cuando la relación de aspecto del archivo de entrada no coincide con la relación de aspecto anchura/altura de destino.
    sample_aspect_ratio ninguno La relación de aspecto de muestra (píxel) que se utilizará para el vídeo de salida. Las opciones de tamaño, ancho, alto y aspect_mode modo hacen referencia a las dimensiones de píxeles codificados, no a las dimensiones de visualización.
    force_interlace_headers falso Algunos decodificadores piden contenido entrelazado, pero jugarán el contenido progresivo muy bien. Esta configuración agregará encabezados «entrelazados» a los metadatos del contenido progresivo, para que parezca que realmente está entrelazado.

    Control de la tasa

    Ajuste Predeterminado Descripción
    quality 3 Seleccione automáticamente la mejor velocidad de bits de vídeo para que coincida con una calidad visual de destino.
    video_bitrate ninguno Una velocidad de bits de vídeo de destino en kbps. No es necesario si selecciona una configuración de calidad, a menos que desee segmentar una velocidad de bits específica.
    audio_quality 3 Seleccione automáticamente la mejor velocidad de bits de audio para que coincida con la calidad de sonido de destino.
    audio_bitrate ninguno Una velocidad de bits de audio de destino en kbps. No es necesario si selecciona una audio_quality configuración, a menos que desee segmentar una velocidad de bits específica.
    max_video_bitrate ninguno Una velocidad de bits media máxima.
    speed 3 Una velocidad de transcodificación objetivo. La codificación más lenta generalmente permite una compresión más avanzada.
    decoder_bitrate_cap ninguno Velocidad de bits máxima alimentada al búfer del decodificador. Normalmente se utiliza para vídeo destinado a streaming o para segmentar dispositivos específicos (p. ej. Blu-Ray).
    decoder_buffer_size ninguno Tamaño del búfer del decodificador, utilizado junto con bitrate_cap.
    one_pass falso Forzar la codificación de una sola pasada.
    audio_constant_bitrate falso Habilite el modo de velocidad de bits constante para audio si es posible.

    Imágenes por segundo

    Ajuste Predeterminado Descripción
    frame_rate La velocidad de fotogramas original (entrada). La velocidad de fotogramas a usar.
    max_frame_rate ninguno La velocidad máxima de fotogramas que se utilizará.
    decimate ninguno Reduzca la velocidad de fotogramas de entrada mediante un divisor.
    keyframe_interval 250 Número máximo de fotogramas entre cada fotograma clave. Para el vídeo H.264, los fotogramas clave son fotogramas IDR.
    keyframe_rate keyframe_interval de 250 El número de fotogramas clave por segundo.
    fixed_keyframe_interval falso Habilite el modo de intervalo de fotogramas clave fijo (sólo VP6, VP8, VP9 y H.264).
    forced_keyframe_interval ninguno Forzar fotogramas clave en el intervalo especificado (sólo H.264).
    forced_keyframe_rate ninguno Especifique el número de fotogramas clave por segundo, teniendo en cuenta la velocidad de fotogramas (sólo H.264).
    generate_frame_index falso Generar un archivo de índice de fotogramas de vídeo
    frame_index_filename ninguno Anule el nombre de archivo predeterminado para el índice de fotograma cuando generate_frame_index esté habilitado.

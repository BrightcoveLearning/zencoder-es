Descripción general

Las solicitudes de API de solo lectura son útiles cuando se realizan solicitudes JSONP sin exponer una clave de API que puede crear nuevos trabajos en su cuenta.

Solicitudes de API compatibles

Las siguientes solicitudes son compatibles con una clave API de solo lectura.

  • Detalles del trabajo
  • Progreso del trabajo
  • Detalles de entrada
  • Progreso de entrada
  • Detalles de salida
  • Progreso de salida

Ejemplos de desinfección

A las solicitudes realizadas con claves de API de solo lectura se les eliminará parte de la información de la respuesta para proteger la información confidencial. Esto incluye, entre otros, URL (en caso de que se proporcione información sobre el nombre de usuario o la contraseña), mensajes de error específicos y otra información potencialmente confidencial.

Devoluciones de llamada JSONP

Para AJAX y otras solicitudes públicas, las claves API de solo lectura pueden acceder a esta información mediante JSONP solicitando la URL con una extensión.js. La respuesta utilizará el progress devolución de llamada a menos que se especifique una usando el callback en la cadena de consulta. Por ejemplo, una solicitud JSONP para el progreso de la entrada sin devolución de llamada especificada:

progress(
  {
      "state": "processing",
      "current_event": "Downloading",
      "progress": "32.34567345"
  }
)

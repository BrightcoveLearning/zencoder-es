Descripción general

Usted especifica la versión de API que desea usar a través de la URL que su aplicación usa para publicar en Zencoder: publicaciones en https://app.zencoder.com/api/v2/ utilizar V2. Si no se especifica una versión, la solicitud usará la API de borde donde las cosas pueden cambiar sin previo aviso, por lo que solo use esto si está experimentando con una nueva función antes de que esté oficialmente disponible, nunca en producción.

Las versiones de la API se cerrarán a medida que aparezcan nuevas versiones en línea, y siempre avisaremos con al menos 6 meses de antelación antes de cerrar una versión para que tenga tiempo de migrar su aplicación.

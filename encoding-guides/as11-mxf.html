Descripción general

Para crear una salida AS-11, configure el format_preset parámetro a "as11_hd". Actualmente, el audio está limitado a audio PCM de 2 canales, con una frecuencia de muestreo máxima de 48 kHz. Para obtener más información sobre las salidas de AVC-Intra, consulte la documentación aquí.

  • format_preset: Ajústelo a as11_hd para la salida ae AVCi 100 MXF.
  • audio_sample_rate: El valor predeterminado no está configurado. Limitado a 48000.

Configuración de ejemplo

Ejemplo de solicitud AS-11

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "format_preset": "as11_hd",
        "audio_sample_rate": 48000
    }
}

Introducción

Al codificar salidas de video para transmisión, no recomendamos usar calidad ajuste solo (esto usa el modo de codificación CRF). Si prefiere utilizar el modo orientado a la calidad, le recomendamos encarecidamente que utilice la configuración de VBV para limitar la variación en la tasa de bits para cumplir con las especificaciones del formato de transmisión.

Configuración de VBV

  • decoder_bitrate_cap
  • decoder_buffer_size

Otras configuraciones recomendadas para considerar

  • video_reference_frames
  • video_bframes

Ejemplo

A continuación, se muestra un ejemplo de codificación orientada a la calidad con restricciones de VBV.

{
    "type":"segmented",
    "width":640,
    "height":360,
    "quality":3,
    "video_codec_profile":"main",
    "video_reference_frames": 4,
    "video_bframes": 3,
    "decoder_bitrate_cap": 1440,
    "decoder_buffer_size": 1920,
    "keyframe_rate": 0.5,
    "max_frame_rate": 30,
    "audio_quality":5,
    "segment_seconds":10
}

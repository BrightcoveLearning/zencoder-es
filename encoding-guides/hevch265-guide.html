Descripción general de HEVC/H.265

HEVC (también conocido como H.265) está disponible para todos los clientes. HEVC es un códec de video de próxima generación que es capaz de ofrecer contenido de mayor calidad a tasas de bits más bajas que su predecesor, H.264.

Recomendaciones de codificación

video_codec_level: restringe la velocidad de bits y las unidades del árbol de codificación (CTU). Lista de valores válidos.

video_codec_profile: establece el perfil de codificación. Los perfiles HEVC admitidos actualmente son: main, main10, main12, main422-10, main422-12, main444-8, main444-10, main444-12. Predeterminado: principal.

video_reference_frames: Limita el número de marcos de referencia para HEVC. Defecto: 3.

video_bframes: Limita el número de bframes consecutivos utilizados para HEVC. Defecto: 3.

tuning: Sintoniza el video de salida para optimizarlo para una prioridad de codificación específica. Tenga en cuenta que para HEVC video, los valores permitidos para tuning están:

  • grano: optimizado para películas con altos niveles de grano.
  • psnr: utiliza una "relación pico de señal a ruido" para optimizar la calidad del video.
  • ssim: utiliza "similitud estructural" para optimizar la calidad del video.
  • fastdecode: reduce la complejidad de la codificación para permitir una decodificación más sencilla.
  • latencia cero: x265 mantendrá un búfer interno de fotogramas para mejorar la calidad; esta configuración elimina ese búfer, pero reduce la calidad.

crf: CRF es una configuración de control de tasa de bits y establece un factor de tasa constante. La configuración de Calidad de Zencoder determina un CRF apropiado para un video determinado. Por lo tanto, si establece tanto Calidad como CRF, la configuración de CRF anulará la Calidad. Los valores válidos son 0-51, siendo los valores más bajos mejor calidad. En términos generales, no debería tener que bajar de 16; 16 es casi sin pérdidas para la mayoría de los archivos. Alrededor de 24 por lo general se ve bastante bien. Alrededor de 35 parece bastante comprimido.

Muestra: HEVC + MP4

A continuación, se muestran configuraciones de muestra para un trabajo que creará una salida HEVC (multiplexada como mp4) con una salida MP4 adicional para clientes que aún no son compatibles con HVEC.

{
  "input": "https://user:pass@bucket-name/file-name.mov",
  "outputs": [
      {
          "label": "hevc",
          "url": "s3://output-bucket/h265.mp4",
          "video_codec": "hevc",
          "video_codec_profile": "main",
          "audio_codec": "aac",
          "format": "mp4",
          "size": "1920x1080"
      },
      {
          "label": "h264",
          "url": "s3://output-bucket/h264.mp4",
          "video_codec": "h264",
          "video_codec_profile": "main",
          "audio_codec": "aac",
          "format": "mp4",
          "size": "640x360"
      }
  ]
}

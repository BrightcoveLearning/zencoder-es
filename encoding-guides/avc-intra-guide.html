Descripción general de AVC-Intra

AVC-Intra define dos preajustes, AVC-Intra 50 y AVC-Intra 100. AVC-Intra 50 está codificado a una velocidad de bits de 50 mbit/s, mientras que AVC-Intra 100 está codificado a 100 mbit/s. Utilizar el video_codec_preset parámetro para definir una salida AVCi. Para obtener información sobre la creación de salidas AS-11, consulte la documentación aquí.

  • video_codec_preset: Establece el preajuste AVC-Intra para las salidas. Valores: avci_50, avci_100.

Configuración de ejemplo

Ejemplo de solicitud de AVC-Intra 50

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_50"
    }
}

Ejemplo de solicitud de AVC-Intra 100

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_100"
    }
}

Crear múltiples salidas

Este tema explica cómo se crean múltiples salidas para un trabajo de codificación.

Crea múltiples salidas

Puede especificar varias salidas pasando más de un hash en la matriz de salidas, así:

{
  "input": "https://user:pass@bucket-name/file-name.avi",
  "outputs": [
    {
      "label": "web",
      "url": "s3://output-bucket/web.mp4",
      "size": "512x384"
    },
    {
      "label": "iphone",
      "url": "s3://output-bucket/iphone.m4v",
      "size": "480x320"
    }
  ]
}

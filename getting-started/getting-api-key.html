Obtener una clave de API

Todas las solicitudes a la API de Zencoder deben autenticarse mediante una clave de API. Este tema explica los diferentes tipos de claves API para Zencoder y cómo obtenerlas.

Obtener una clave de API

Para solicitar una clave API, vaya a el sitio de Zencoder. Si ya se ha registrado como usuario, inicie sesión. De lo contrario, regístrese como nuevo usuario; no puede solicitar claves API hasta que esté registrado como usuario.

Una vez que haya iniciado sesión, vaya a la sección API:

En la sección API, verá opciones para solicitar diferentes tipos de claves API:

Consulte las secciones siguientes para obtener una explicación de los diferentes tipos de claves API. Tenga en cuenta que debe estar registrado en un plan pago para obtener las claves de acceso completo.

Claves de acceso completo

Las claves de acceso completo permiten realizar cualquier tipo de solicitud a la API de Zencoder. Este es el tipo de clave que necesitará para enviar trabajos a Zencoder y obtener resultados.

Leer solo claves

Las claves de acceso de solo lectura le permiten hacer GET solicitudes solo a la API de Zencoder. Son útiles para los usuarios que solo necesitan obtener el estado de los trabajos, pero que en realidad no están enviando trabajos. Ver Usar claves de API de solo lectura para más información.

Teclas del modo de integración

Las teclas del modo de integración le permiten enviar trabajos, pero los trabajos no se procesarán. Son útiles principalmente para integradores de sistemas que desean probar solicitudes en busca de errores. También se pueden usar para familiarizarse con el uso de la API antes de comprar un plan pago de Zencoder, o para verificar solicitudes antes de obtener resultados.

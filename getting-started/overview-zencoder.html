Descripción general: Zencoder

Zencoder es una poderosa plataforma para transcodificar videos bajo demanda.

Introducción

Zencoder ofrece un servicio de transcodificación rápido y confiable para archivos de video y audio. Puede producir resultados para la web y una amplia variedad de dispositivos, y puede integrarse fácilmente con su CMS y otras aplicaciones.

La interfaz de usuario

Zencoder es principalmente una API, pero hay una interfaz de usuario que incluye:

  • Configuraciones de la cuenta
  • Un panel que muestra los resultados de trabajos recientes.
  • Un generador de solicitudes de API que le permite comenzar con la API sin escribir código

La API

Hay varias funciones que puede realizar usando la API, la más importante de las cuales es enviar y administrar trabajos de codificación para video bajo demanda.

Para obtener una guía completa de los puntos finales de la API, consulte la Referencia de API.

El poder de Zencoder radica en la flexibilidad que le ofrece para crear exactamente el tipo de salidas que desea. Esta también es la fuente de cierta complejidad, ya que hay muchas configuraciones de codificación para comprender. Un buen lugar para empezar a explorar estos datos es el resumen de la configuración de codificación, que contiene enlaces que le permiten acceder a las descripciones completas con código de muestra.

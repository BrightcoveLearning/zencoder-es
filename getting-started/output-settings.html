Ajustes de salida

Este tema explica las salidas predeterminadas para trabajos de Zencoder y los conceptos básicos para definir sus propias salidas.

Definición de salidas

Nuestra salida predeterminada es un solo archivo de video con la siguiente configuración:

Vídeo H.264, audio AAC, formato de archivo MP4
Video: calidad media
Audio: estéreo, calidad media (aproximadamente 112 kbps), 44100 Hz

Si no especifica una ubicación de salida, el archivo será alojado temporalmente por Zencoder. Después de 24 horas, el archivo ya no estará disponible.

Por supuesto, puede controlar el ajustes de salida. Para hacer esto, pase una serie de opciones en la solicitud de API, como esta:

{
  "api_key": "93h630j1dsyshjef620qlkavnmzui3",
  "input": "s3://bucket-name/file-name.avi",
  "outputs": [
    {
      "url": "s3://output-bucket/output-file-name.mp4",
      "width": "1280",
      "height": "720"
    }
  ]
}

En este ejemplo se creará un único archivo de salida a partir del vídeo de entrada original y lo cargará en un bucket de S3. Puedes crear múltiples archivos de salida del original sin tener que crear trabajos de codificación adicionales.

Significado

Este error se produce porque el archivo enviado está encriptado, lo que nos impide abrirlo. Por lo general, esto significa que un archivo tiene cifrado DRM, generalmente porque el archivo se compró a través de una fuente como iTunes.

¿Reintentar este trabajo?

Trabajos con este error no debe volver a intentarlo. Obtenga más información sobre cómo reintentar trabajos.

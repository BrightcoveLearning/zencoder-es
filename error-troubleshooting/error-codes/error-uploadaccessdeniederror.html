Significado

Intentamos subir el archivo a la dirección especificada, pero obtuvimos un error de permiso. Si se trata de una URL de S3, probablemente necesite conceder acceso de lectura a Zencoder para este depósito o archivo. La forma más sencilla de hacerlo es utilizar un cliente S3, como la consola AWS de Amazon.

En términos técnicos, esto significa que obtuvimos una respuesta 403 Acceso denegado del servidor.

Soluciones alternativas

Asegúrese de que los permisos en el archivo en la URL proporcionada permitan el acceso a Zencoder. Si el archivo está en S3, es posible que deba conceder acceso de lectura a Zencoder.

¿Reintentar este trabajo?

Los trabajos con este error se pueden reintentar. Obtenga más información sobre cómo reintentar trabajos.

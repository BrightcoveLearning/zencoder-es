Significado

Este error ocurre cuando el medio de entrada no es compatible con la generación de perfiles dinámicos CAE. Esto podría significar que el archivo multimedia de entrada solo tiene datos de audio y / o imagen y no contiene ninguna pista de video. Póngase en contacto con el soporte de Zencoder si desea más información.

¿Reintentar este trabajo?

Trabajos con este error no debe volver a intentarlo. Obtenga más información sobre cómo reintentar trabajos.

Significado

Zencoder admite cientos de códecs, pero todavía hay algunos que no admitimos:

  • Intermedio de manzana
  • HDV 720p60
  • Go2Meeting3 / 4 (G2M3 / G2M4)

Si envía estos archivos, fallarán. La razón de esto es que nuestros trabajadores de transcodificación son todos Linux y no hay un decodificador de Linux para estos códecs. Estamos trabajando activamente para agregar soporte para estos códecs, probablemente mediante la adición de servidores que no son Linux.

¿Reintentar este trabajo?

Trabajos con este error no debe volver a intentarlo. Obtenga más información sobre cómo reintentar trabajos.

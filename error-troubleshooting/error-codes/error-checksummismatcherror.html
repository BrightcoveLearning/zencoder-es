Significado

La suma de comprobación MD5 calculada para el archivo de entrada no coincide con la suma de comprobación proporcionada a la API.

Hay una variedad de posibles causas para este problema. Los más probables son:

  • El archivo se corrompió en tránsito.
  • La suma de comprobación proporcionada a la API era incorrecta.

Soluciones alternativas

Verifique para asegurarse de que la suma de verificación que proporcionó a la API coincida con la suma de verificación del archivo en el disco.

Si ellos no coincidencia, realice una nueva llamada a la API con la suma de comprobación adecuada.

Si las sumas de control hacer coincidencia, es posible que el archivo se haya dañado durante la transferencia. Vuelva a enviar el trabajo.

En cualquier caso, si el reintento falla, comuníquese con nosotros.

¿Reintentar este trabajo?

Los trabajos con este error se pueden reintentar. Obtenga más información sobre cómo reintentar trabajos.

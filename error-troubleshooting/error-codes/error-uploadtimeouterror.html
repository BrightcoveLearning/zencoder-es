Significado

Intentamos cargar un archivo en la dirección especificada, pero el servidor en esa dirección tardó demasiado en responder. Intentamos descargar el archivo 3 veces para asegurarnos de que el problema no sea un problema de conectividad temporal.

¿Reintentar este trabajo?

Trabajos con este error no debe volver a intentarlo. Obtenga más información sobre cómo reintentar trabajos.

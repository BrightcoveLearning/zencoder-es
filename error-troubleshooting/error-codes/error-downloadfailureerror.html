Significado

Este error ocurre cuando no podemos descargar un archivo por un motivo que no sea Acceso denegado o Archivo no encontrado. Si el servidor devuelve un código de respuesta, lo mostraremos en el mensaje de error para que pueda determinar qué sucedió y realizar las actualizaciones necesarias en el archivo.

¿Reintentar este trabajo?

Trabajos con este error no debe volver a intentarlo. Obtenga más información sobre cómo reintentar trabajos.

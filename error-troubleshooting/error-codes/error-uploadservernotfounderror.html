Significado

Intentamos conectarnos al servidor en la dirección especificada, pero ningún servidor respondió. Esto puede suceder si el servidor está detrás de un firewall, si el acceso está restringido a cierta dirección IP o si el servidor está fuera de línea.

Soluciones alternativas

Asegúrese de que el servidor esté en línea y sea de acceso público para solicitudes externas. Si el acceso al servidor está restringido a ciertas IP, puede obtener más información sobre la Rangos de IP que utiliza Zencoder.

¿Reintentar este trabajo?

Los trabajos con este error se pueden reintentar. Obtenga más información sobre cómo reintentar trabajos.

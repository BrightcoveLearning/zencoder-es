Significado

Intentamos descargar un archivo en la dirección especificada, pero en lugar de un archivo de video o audio, obtuvimos algo más, como un documento HTML o XML. Esto podría ser algo así como una página de error de Archivo no encontrado, pero estas páginas generalmente regresan con un código de respuesta 404 (o algún otro código de error), y este error solo ocurre cuando el código de respuesta HTTP era 2XX, lo que indica que el sitio web pensó esto fue una solicitud exitosa. Por lo tanto, puede ser un error tipográfico, una URL de entrada incorrecta o una página de error mal configurada.

¿Reintentar este trabajo?

Trabajos con este error no debe volver a intentarlo. Obtenga más información sobre cómo reintentar trabajos.

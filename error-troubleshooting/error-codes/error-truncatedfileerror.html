Error: Error de archivo trunqueado

Este tema proporciona detalles sobre el error de la API de Zencoder: TruncatedFileError

Significado

El archivo de entrada es más pequeño de lo que los encabezados dicen que debería ser. Generalmente, esto significa que el archivo se cortó en algún momento antes de ser transferido a Zencoder.

¿Reintentar este trabajo?

Trabajos con este error no debe volver a intentarlo. Obtenga más información sobre cómo reintentar trabajos.

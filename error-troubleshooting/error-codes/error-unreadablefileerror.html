Significado

Este error ocurre cuando no podemos decodificar un archivo multimedia. Por lo general, significa que el archivo en sí está dañado y no se puede decodificar. Aceptamos casi todo, por lo que generalmente esto indica un archivo corrupto.

Ocasionalmente, significa que el archivo usa un códec de entrada que no admitimos y que no hemos visto antes. (Las fallas de códec conocidas obtienen un UnsupportedCodecError.) De cualquier manera, veremos si hay algo que podamos hacer para que el archivo funcione. A veces podemos arreglar la corrupción o agregar soporte de códec, pero a veces el archivo está irreversiblemente dañado. En cualquier caso, siéntete libre de Contáctenos para obtener detalles específicos.

¿Reintentar este trabajo?

Trabajos con este error no debe volver a intentarlo. Obtenga más información sobre cómo reintentar trabajos.

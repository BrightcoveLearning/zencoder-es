Significado

Este error ocurre cuando hay un problema durante la generación del perfil dinámico CAE. Esto generalmente significa que el perfil no se puede generar de una manera que cumpla con las limitaciones especificadas (límites de reproducción, límites de velocidad de bits, límites de granularidad, etc.). Póngase en contacto con el soporte de Zencoder si desea más información.

¿Reintentar este trabajo?

Trabajos con este error no debe volver a intentarlo. Obtenga más información sobre cómo reintentar trabajos.
